About Melbourne AI

What is Melbourne AI Video Creator?

Melbourne AI is an open-source, automated video generation system that transforms text prompts into professional-quality videos. It combines multiple AI models to create unique visuals, animations, voiceovers, and automatic editing—all without requiring paid APIs or cloud services.

⚙️ How It Works

  1. Script Generation: AI creates a detailed video script from your prompt
  2. Image Generation: Unique images are created for each scene using Stable Diffusion
  3. Animation: Static images are animated with effects like Ken Burns and fades
  4. Voiceover: Natural narration is generated using Text-to-Speech
  5. Video Editing: All elements are combined into a professional final video

🔧 Technology Stack

  • Python (FastAPI, MoviePy, FFmpeg)
  • Stable Diffusion (AI Image Generation)
  • Coqui TTS (Text-to-Speech)
  • Ollama/LM Studio (LLM Integration)
  • Flask (Web Framework)

💰 Cost

100% Free! All tools are open-source and self-hosted. No API fees, no cloud costs.
